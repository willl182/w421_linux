# Technical Finding / Milestone

**Context**: OpenClaw Docker configurado en modo solo Z.AI GLM 4.7

## The Discovery

- El flujo mas estable para configurar modelos fue por CLI con `openclaw config set`, evitando errores de parser en la UI.
- Se confirmo contenedor activo `openclaw-hacw-openclaw-1` y se aplicaron cambios de modelo sin tocar secretos.
- `primary` y sesion `main` quedaron apuntando a `zai/glm-4.7`, con `fallbacks` vacio.
- El comando correcto de listado es `openclaw models list` (plural).
